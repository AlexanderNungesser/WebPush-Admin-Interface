Create a simple sitemap

Want a sitemap for your page to ease navigation and overview for your visitors? Simply follow the following steps:

  1. Configure your page menue by editing the routes.js
  2. Clear the cache for this site
  3. Reload this page
  4. Click the "generate code" button
  5. Copy code to clipboard
  6. Open this file (sitemap.html) in editor
  7. Paste the generated code
  8. Outcomment this todo and the component useage.
